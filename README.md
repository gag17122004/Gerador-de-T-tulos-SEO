# 📈 Gerador de Títulos SEO - PRATIK Distribuidora

Repositório estratégico para criação e otimização de títulos de produtos. Focado em maximizar o alcance orgânico e a taxa de clique (CTR) nos algoritmos da **Shopee, Mercado Livre e SHEIN**.

---

## 🎯 Objetivo
Transformar características técnicas (filtros, purificadores, envelopes e sacos de lixo) em títulos magnéticos que respeitem os limites de caracteres e as palavras-chave mais buscadas do setor.

## 🧠 Lógica de Construção (Framework PRATIK)
Nossos títulos seguem a hierarquia:
**[Produto Principal] + [Atributo de Diferenciação] + [Especificação Técnica] + [Benefício/Uso]**

---

## 📂 Organização do Projeto

| Pasta | Conteúdo |
| :--- | :--- |
| `📂 templates/shopee` | Foco em palavras-chave repetitivas e termos de busca direta. |
| `📂 templates/mercadolivre` | Estruturas limpas (Sem "Frete Grátis" no título) para melhor indexação. |
| `📂 templates/shein` | Títulos curtos e visuais, focados no padrão internacional da plataforma. |
| `📂 keywords` | Planilhas e listas de termos extraídos do Planejador de Palavras do Google e Ads dos Marketplaces. |

---

## 🛠️ Como Utilizar

1. **Consulta de Palavras-Chave:** Acesse `/keywords` para identificar o termo principal (ex: "Refil Filtro").
2. **Escolha do Template:** Selecione o arquivo de template conforme o marketplace de destino.
3. **Validação de Tamanho:**
   - **Shopee:** Máx. 255 caracteres.
   - **Mercado Livre:** Máx. 60 caracteres (Otimizado).
   - **SHEIN:** Máx. 100 caracteres.
4. **Aplicação no Tiny ERP:** Após gerar o título, atualize no cadastro do Tiny para sincronização via API.
